Metodos e propriedades!

Métodos

Métodos são ações que um objeto pode realizar. No mundo real, um celular pode realizar várias ações, como ligar, desligar, tocar música e tirar fotos..

Exemplo: celular

Se tivermos um objeto chamado CELULAR, ele pode ter métodos como:

celular.ligar() → Liga o celular.

celular.desligar() → Desliga o celular.

celular.tirar_foto() → Tira uma foto.

celular.reproduzir_musica() → Reproduz música.

*Como isso funciona no código?

No JavaScript, podemos criar esse objeto assim:

Criamos uma classe chamada Celular, que define os métodos.

Os métodos são ações que o celular pode fazer, como ligar, desligar, tirar fotos e tocar música.

Criamos um objeto meuCelular e chamamos os métodos com meuCelular.metodo().

Propriedades

Propriedades do objeto

Um carro real tem propriedades como peso e cor:

car.name = Fiat, car.model = 500, car.weight = 850kg, car.color = branco.

Objetos de carro têm as mesmas propriedades , mas os valores diferem de carro para carro.
